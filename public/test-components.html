Phase 1에서 구축한 5개 공통 컴포넌트의 동작을 확인하는 페이지입니다.

1 이미지 압축 image-compressor.js

이미지를 선택하면 500KB 이하 · 긴 변 1280px로 자동 압축합니다.

압축 중...

이미지를 클릭하면 뷰어로 열립니다.

2 이미지 뷰어 image-viewer.js

썸네일 클릭 → 풀스크린 → 휠/핀치 줌, 드래그 이동 → 이미지를 한 번 더 누르거나 ESC로 닫기.

3 외부 링크 렌더링 link-renderer.js

본문 내 URL이 자동으로 새 창 외부 링크로 변환됩니다 (noopener noreferrer).

렌더링 결과 ↓

4 댓글 스레드 comment-thread.js

현재 사용자: 테스터(담당자) — 본인 댓글만 수정·삭제할 수 있습니다.

5 게시글 작성자 액션 post-actions.js

작성자 본인 또는 관리자에게만 수정·삭제 메뉴(⋯)가 표시됩니다.
